# PC Equivalence APP

This is a very simple app that calculates how many of your very first PC's it would take to power your modern PC.

Written in Python with Streamlit

## Run Locally

1. Make sure you have Python installed.
2. Install Streamlit (I recommend using a venv):

    ```bash
    pip install streamlit
    ```

    Or, use the included requirements.txt

    ```bash
    pip install -r requirements.txt
    ```

3. Start the app from the project folder:

    ```bash
    streamlit run pc_equivalence_app.py
    ```

4. Streamlit will open the app in your browser, usually at `http://localhost:8501`.
